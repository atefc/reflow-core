# AI Prompts Used in Reflow Core Project

## Explain the requested work with examples
**Prompt:** I have this task to implement; explain how the system works with a simple scenario.

## Generate data for different scenarios
**Prompt:** Create data to test my work under various scenarios:  
- Without conflicts  
- With conflicts on the same work center  
- With conflicts across different work centers  
- With maintenance overlaps

## Review DAG algorithm with examples
**Prompt:** I want to implement DAG principles; explain how it works with simple examples.

## Generate Jest tests for utility functions
**Prompt:** Generate basic Jest tests for these functions.
